# JS Multiplayer Sandbox Game

The base of a Multiplayer Sandbox Game with Server reconciliation, Client prediction and Entity Interpolation.
There is a lot of work to do but this is a base functional code.

### Technologies

- Typescript
- Express
- Websockets
- HTML Canvas
